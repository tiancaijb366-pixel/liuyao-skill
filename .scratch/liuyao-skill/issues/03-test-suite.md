# 03 — 卦例测试套件

**What to build:** 从三本参考书中提取 25-36 个结构化卦例，编写测试套件，验证 agent 解读结论与原作的一致性，目标通过率 ≥80%。

**Blocked by:** 02 — 规则表附录完整嵌入（需要完整 SKILL.md 才能进行有意义的测试）

**Status:** ready-for-agent

## 具体内容

### 测试用例来源

| 来源 | 数量 | 覆盖范围 |
|------|------|---------|
| 增删卜易 | 15-20 例 | 用神章、元神忌神章、各门类（占财/占病/占功名/占出行/占婚姻） |
| 易冒 | 5-8 例 | 类总章、合冲章、墓绝章、长生章等特色理论 |
| 古筮真诠 | 5-8 例 | 心态卦、静卦、真假用神、暗动等 |

### 测例格式

```json
{
  "id": "zs-001",
  "来源": "增删卜易·用神章",
  "问题": "占父近病",
  "时间": "辰月戊申日",
  "本卦": "乾为天",
  "变卦": "风天小畜",
  "世应": "世在上爻 应在三爻",
  "动爻": "午火动化未土",
  "日月建": "辰月 戊申日",
  "野鹤断语": "近病逢冲即愈",
  "野鹤用神": "辰土父母",
  "应期": "丑日",
  "验证点": ["用神选取", "六冲断法", "贪合忘克", "应期判断"]
}
```

### 验证指标

| 指标 | 目标 | 方法 |
|------|------|------|
| 吉凶结论一致率 | ≥80% | agent Phase 1 结论 vs 原作断语 |
| 用神选取正确率 | ≥85% | agent 用神 vs 原作用神 |
| 应期范围匹配率 | ≥75% | 允许 ±1 时间单位误差 |
| 关键因素检出率 | ≥80% | agent 分析中是否覆盖原作提到的关键因素 |

### 测试执行

- 每个测例跑一次 agent
- 检查 `.scratch/liuyao-skill/analysis/` 内部分析文件
- 逐项比对并记录通过/失败
- 输出测试报告

## 验收标准

- [ ] 25-36 个测例已提取完成，JSON 格式
- [ ] 测试脚本/流程可执行
- [ ] 第一次测试运行完成，生成测试报告
- [ ] 按测例类型（增删/易冒/古筮真诠）统计通过率
- [ ] 标注每个失败测例的原因（如"用神取错"、"未识别六冲"等）
