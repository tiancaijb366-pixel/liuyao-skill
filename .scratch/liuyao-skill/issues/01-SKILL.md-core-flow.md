# 01 — SKILL.md 核心流程

**What to build:** 编写六爻占卜解读 skill 的主流程文件 SKILL.md，包含完整的 11 步解读流程（易冒分层下钻为骨架，嵌入古筮真诠的对轨和增删的四力框架）和基础规则表。agent 加载后能按结构化解读者提供的卦象数据。

**Blocked by:** None — can start immediately.

**Status:** ready-for-agent

## 具体内容

### 1. 11 步解读流程

```
① 解析数据（识别本卦/变卦/六亲/六神/世应/动爻/日月建）
② 辨卦（八种卦用：内外/冲合/反伏/墓绝/归游/卦候/卦义/卦验）
③ 实证对轨 ← 暂停点，问用户确认现状 + 来意 + 判断事态卦/心态卦
④ 辨爻（六种爻用：世应/两身/六亲/六神/爻位/变互）
⑤ 辨卦种（静卦→特殊公式 / 动卦→正常路线）
⑥ 世应十忌（空破绝散墓动合冲随墓助伤升降进退）
⑦ 定用神 + 四处生克冲合（事态卦/心态卦双路线）
⑧ 专项检查（动变/飞伏/日月/合冲/空破/刑害/六神/暗动）
   【贯穿两个分析层面：吉凶判断层 vs 应期细节层】
⑨ 用神十八法（量化评分：全吉8→半吉3→凶陷7）
⑩ 应期（克期之法 + 应期公式）
⑪ 趋吉避凶（建议）
```

### 2. 对外输出策略

- 内部分析写入 `.scratch/liuyao-skill/analysis/{timestamp}-analysis.md`
- 对外只输出 Phase 1（核心结论 + 卦象总览），之后纯追问驱动
- 无 Phase 2/3/4 预设展开

### 3. 基础规则表（内联嵌入正文）

- 六亲排法（生我者父母 / 我生者子孙...）
- 用神分类表（扩展现有 SKILL.md 用神表）
- 进神退神规则
- 应期规则表

### 4. 规则表附录占位
