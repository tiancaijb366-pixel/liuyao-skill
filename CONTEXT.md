# 六爻占卜解读 Skill

基于《增删卜易》《易冒》两本参考书输出有据可查的六爻解读。

## 核心输入

用户提供的卦象数据需包含以下六个字段，自由格式粘贴，skill 自行解析：

- **本卦/变卦** — 主卦和变卦的卦名（如火风鼎、山风蛊）
- **世应** — 世爻和应爻的位置（如世在二爻、应在五爻）
- **六亲** — 各爻的六亲属性（父母、兄弟、官鬼、妻财、子孙）
- **六神** — 各爻的六神（青龙、朱雀、勾陈、螣蛇、白虎、玄武）
- **动爻** — 哪一爻发动、如何变化（如三爻酉金动化戌土）
- **日月建** — 占卜时的月支和日支（如申月戊午日）

## Language

**本卦**:
起卦所得的原卦，代表事物的初始状态。重卦由下（内）上（外）两单卦组成，下三爻为下卦，上三爻为上卦。

**变卦**:
因动爻而变出的卦，代表事物的发展结果。本卦动爻阳变阴、阴变阳后得到变卦。

**世应**:
世爻代表占卜者（自己），应爻代表所问之事或对方。世应在卦中的位置由卦宫和爻位决定。断卦时世应生克关系是核心参考之一。

**用神**:
六爻解读的核心。根据所问之事选取对应的六亲爻：占父母以父母爻为用神，占财以妻财爻为用神，占功名以官鬼爻为用神，占子孙以子孙爻为用神，占兄弟以兄弟爻为用神。用神旺衰吉凶直接决定占卜结果。
_参考_: 《增删卜易》用神章

**元神**:
生用神之爻。元神旺相则用神有源，吉。元神休囚或被克则用神无助。

**忌神**:
克用神之爻。忌神旺动克用神为凶。忌神休囚安静或受制则凶不成。

**仇神**:
生忌神之爻。仇神旺则忌神有力克用神，间接为凶。

**月建**:
占卜之月的月支。月建能生克冲合卦中诸爻，为万卜之提纲。爻临月建为旺相，月建克之为休囚。
_参考_: 《增删卜易》月将章

**日辰**:
占卜之日的日支。日辰为六爻之主宰，能生克冲合卦中诸爻，也能冲实旬空、冲散月破。日辰力量与月建等同。
_参考_: 《增删卜易》日辰章

**动爻**:
卦中发动之爻（标记为○或×）。动爻是断卦的关键，动则有变，变爻亦参与生克。独发之爻往往为吉凶之枢机。

**变爻**:
动爻所变出的爻（本卦阴变阳、阳变阴后得到）。变爻可生克本卦诸爻，回头生克直接影响动爻本身。

**旬空**:
天干地支配合中，每旬有两位地支为空亡。爻值旬空则暂时无力，出空后恢复。用神逢空为大忌，但近病逢空反吉。
_参考_: 《增删卜易》旬空章

**月破**:
爻被月建所冲则为月破。月破之爻休囚无力，逢日辰合或填实才能发挥作用。

**暗动**:
静爻被日辰冲则为暗动。暗动之爻虽不显于卦面，但暗中有力，往往对应隐藏或未明之事。
_参考_: 《增删卜易》暗动章

**六冲**:
地支子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲。卦逢六冲主快、散、不和。六冲变六合为先冲后合，六合变六冲为先合后散。
_参考_: 《增删卜易》六冲章

**六合**:
地支子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合。卦逢六合主慢、聚、和合。用神逢合为吉事有助。

**三合局**:
申子辰合水局、亥卯未合木局、寅午戌合火局、巳酉丑合金局。三合局力量强大，吉凶倍增。

**回头生/回头克**:
动爻化出之爻（变爻）生该动爻为回头生，克该动爻为回头克。回头生为吉，回头克为凶。如金动化土为回头生，金动化火为回头克。
_参考_: 《增删卜易》动变章

**进神/退神**:
爻动化进（如寅化卯）为力量增强，爻动化退（如卯化寅）为力量衰退。进神主事之渐进，退神主事之渐消。
_参考_: 《增删卜易》进神退神章

**飞伏神**:
当用神不上卦时，需在本宫首卦中寻找隐藏的用神爻，称为伏神；卦面上显现的对应爻位称为飞神。飞来生伏为吉，伏来克飞为凶。
_参考_: 《增删卜易》飞伏神章

**独发/独静**:
卦中仅一爻发动为独发，仅一爻安静为独静。独发之爻往往是吉凶的关键指示。
_参考_: 《增删卜易》独发章

**随鬼入墓**:
官鬼爻入墓（辰戌丑未）。世爻随官鬼入墓为凶兆，占病尤其忌讳。
_参考_: 《增删卜易》随鬼入墓章

**生旺墓绝**:
五行在十二宫中的状态：长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养。断卦常用：
- 长生：初生有力
- 帝旺：力量鼎盛
- 墓：收藏无力
- 绝：灭绝无气
_参考_: 《易冒》墓绝章第十八

**应期**:
应验的时间。用神旺相则速应期近，用神休囚则缓应期远。常用的应期判断方法：用神逢值（临值之日/月）、用神逢合、用神逢冲（旬空填实）、忌神逢制之时。
_参考_: 《增删卜易》各门类应期总注章

**纳甲**:
将十天干和十二地支纳入八卦的系统。乾纳甲壬、坤纳乙癸、震纳庚、巽纳辛、坎纳戊、离纳己、艮纳丙、兑纳丁。地支纳于各爻，用于定六亲、六神和生克关系。
_参考_: 《易冒》纳甲章第四

**六神**:
青龙（主吉、婚姻、财喜）、朱雀（主口舌、文书、信息）、勾陈（主田土、迟滞）、螣蛇（主虚惊、怪异）、白虎（主血光、丧服、旺相为武事）、玄武（主暗昧、盗贼、隐私）。六神配日辰天干定位。
_参考_: 《增删卜易》六神章

**归魂/游魂**:
八宫卦中，归魂卦主归、游魂卦主变。占出行得游魂主不定，占归来得归魂主归期将至。

**中间分析文件**:
Phase 2 的系统性扫描结果写入的临时文件（`.scratch/liuyao-skill/analysis/`），agent 内部使用，不持久化。

**隐式引用**:
引用不出现在解读输出中。当用户追问来源时，agent 从参考书中调出对应章节。保持解读的叙事流畅性。

## 参考书

- 《增删卜易》（清·野鹤老人著/李文辉增删）— 主流程骨架，卦例最丰富
- 《易冒》（清·程良玉著）— 系统性检查清单，理论框架完整
