# 六爻占卜解读 Skill

基于《增删卜易》《易冒》的六爻结构化解读流程，支持多 Agent 使用。提供从排盘数据到完整解读的四阶段式流程，覆盖用神、元神忌神、动变、日月建、旬空、六冲、六神、三合等核心概念。

## 安装

### 通过 pi 安装（推荐）

```bash
pi install git:github.com/tiancaijb366-pixel/liuyao-skill
```

安装后可在 pi 中直接调用 skill：

> 我正在用六爻占卜，帮我解读这个卦象数据：...

### 手动安装

| Agent | 安装方式 |
|-------|---------|
| **pi / Workbuddy** | 将本仓库复制到 `~/.pi/agent/skills/liuyao-skill/` |
| **Claude Code** | 将 `SKILL.md` 内容附加到项目根目录的 `CLAUDE.md`，或使用 `/skill:` 命令引用 |
| **Codex CLI** | 将 `SKILL.md` 内容复制到 `CODEX.md` |
| **Cursor** | 引用 `SKILL.md` 内容到 `.cursorrules` |
| **其他 Agent** | 复制 `SKILL.md` 内容作为 system prompt 的一部分 |

## 快速开始

### 前置准备

使用前需要准备六爻卦象数据。推荐使用以下在线排盘工具获取：

- [astro-seek 六爻排盘](https://horoscopes.astro-seek.com/iching-divination) — 最推荐，数据完整，适合本 skill
- 其他排盘工具（只需输出包含本卦、变卦、世应、动爻、日月建等信息的格式）

### 使用流程

**Step 1**：获取排盘数据。例如：

```
时间：2024-01-15 14:30
本卦：火风鼎
变卦：火山旅
世应：世在初爻，应在四爻
动爻：初爻、三爻动
日建：甲子
月建：乙丑
旬空：寅卯
六冲：否
```

**Step 2**：将数据提供给 agent，触发 skill：

> 帮我解读这个六爻卦：本卦火风鼎，变卦火山旅，世在初爻应在四爻，初爻三爻动，日建甲子月建乙丑，旬空寅卯...

**Step 3**：Agent 会按四阶段流程自动解读：

| 阶段 | 内容 |
|------|------|
| **Phase 1 — 核心三角** | 用神定位 → 日月建影响力 → 动变关系 |
| **Phase 2 — 整体扫描** | 特殊格局（六冲/六合/伏吟/反吟等）→ 五行力量对比 |
| **Phase 3 — 展开解读** | 用神状态判断 → 吉凶定性 → 应期推断 |
| **Phase 4 — 整合收尾** | 综合结论 → 建议 → 注意事项 |

### 示例

输入：

```
时间：2024-06-20 10:00
本卦：水山蹇
变卦：地山谦
世应：世在五爻，应在二爻
动爻：五爻动
日建：庚申
月建：戊午
旬空：子丑
```

Agent 会输出类似以下的完整解读：

> **【用神】** 占问事业，以官鬼爻为用神。本卦水山蹇五爻申金妻财动化亥水子孙，用神官鬼不上卦，伏于二爻午火之下。
>
> **【日月建】** 日建庚申金生用神（伏神午火），月建戊午火为用神帝旺之地，日月皆助。
>
> **【动变】** 五爻申金动化亥水，申金生伏神午火，亥水克之。动而化出之亥水与月建午火六合，力有牵制。
>
> **【吉凶判断】** 用神得日月生扶，虽伏藏而有气。五爻动来生用神，本为吉兆。但化出亥水与月建合，吉中藏变。初吉终吝。
>
> **【应期】** 用神午火伏藏，待飞神（亥水）逢制或伏神值临之时应事。秋金旺制亥水，或午月伏神临值，为应期窗口。
